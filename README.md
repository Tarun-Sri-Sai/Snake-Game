# Snake Game

This repository contains a simple terminal-based Snake Game written in C++.

## How to Play

To play the game, simply navigate to `Snake-Game` folder and type:

 ```make run```

This will run the `game.exe` file located in the `bin` directory. The game will start and you can use the arrow keys to move the snake around the board. The objective of the game is to eat as many apples as possible without running into the walls or the snake's own body. If the snake collides with either of these, the game will end.

## Compilation

To compile the source code, use the provided `Makefile` by running the following command in the terminal:

```make```
